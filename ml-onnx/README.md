# Rust Notes

Install: `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`

Update: `rustup update`

Initialize Directory: `cargo init postgresql-geospatial`

Add dependency: `cargo add sqlx@=0.8.6`

Commands:
```
cargo run
cargo build
cargo clean
cargo build --release
```
